# flexget-plugins-ruvoice

**flexget-plugins-ruvoice** is the bundle of search/urlrewrite pugins for FlexGet.

---

## LostFilm

Web site: [lostfilm.tv](http://lostfilm.tv)

### Configuration

#### Authorization

```yaml
lostfilm_auth:
  username: 'username_here'
  password: 'password_here'
```
